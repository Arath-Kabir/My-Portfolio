## 🖼️ Adding Images

All images must be placed inside:

```

assets/images/

```

Use **exact file names** as referenced in `index.html`.

### Example:
```

profile.jpg
project-1.jpg
certificate-1.jpg
activity-1.jpg
gallery-1.jpg

```

⚠️ Important:
- Use lowercase file names
- Avoid spaces (use `-` instead)
- Match `.jpg` or `.png` exactly

## 🚀 Deployment (GitHub Pages)

### Step 1: Create Repository
- Go to GitHub
- Create a new repository (e.g., `portfolio`)

### Step 2: Upload Files
Upload:
- `index.html`
- `assets/` folder
- `README.md`

### Step 3: Enable GitHub Pages
- Go to **Settings → Pages**
- Source: `main` branch
- Folder: `/ (root)`
- Click Save

### Step 4: Access Website
GitHub will generate a link like:
```

[https://your-username.github.io/portfolio/](https://your-username.github.io/portfolio/)

```
